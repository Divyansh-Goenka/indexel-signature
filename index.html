How to add in Outlook
  1. Click Copy Signature above
  2. Open Outlook → File → Options
  3. Go to Mail → Signatures
  4. Click New, give it a name
  5. Paste with Ctrl + V
  6. Click OK to save
How to add in Gmail
  1. Click Copy Signature above
  2. Open Gmail → Settings ⚙️ → See all settings
  3. Scroll to Signature → Create new
  4. Paste with Ctrl + V
  5. Scroll down and click Save Changes
